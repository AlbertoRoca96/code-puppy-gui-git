# Store Assets Checklist

## Shared assets
- App name and short description
- Full description / marketing copy
- Privacy policy URL
- Support URL
- Contact email
- Primary app icon source files
- Release notes template

## Apple App Store / TestFlight
- iPhone screenshots
- iPad screenshots if you ever enable tablet support
- App preview video (optional, but nice)
- Promotional text
- Keywords
- Support URL
- Marketing URL
- Privacy policy URL
- App privacy nutrition labels completed in App Store Connect
- Age rating questionnaire completed

## Google Play Store
- 512x512 app icon
- Feature graphic
- Phone screenshots
- 7-inch / 10-inch tablet screenshots only if relevant later
- Short description
- Full description
- Privacy policy URL
- Contact details
- Data safety form completed
- Content rating completed

## Reality check
CI can build binaries.
CI cannot invent screenshots, marketing copy, privacy answers, or your will to fill out store forms.
