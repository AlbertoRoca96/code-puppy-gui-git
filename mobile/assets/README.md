# App Icons

Add your app icons here:

- `icon.png` - Main app icon (1024x1024 PNG)
- `adaptive-icon.png` - Android adaptive icon (foregound 1024x1024 PNG)

Get free icons from: https://expo.dev/pwn

For the Code Puppy app, use a cute puppy emoji or illustration.

Example:
- Use 🐶 emoji with a gradient background
- Or a stylish puppy illustration

The icon is referenced in app.config.ts.
